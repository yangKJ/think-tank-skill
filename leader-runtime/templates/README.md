# templates

`leader-runtime/templates/` 存放领导者运行层专属模板。

这里的模板不属于 `think-tank` 高阶 Skill 输出，而属于：

- leader 派遣
- expert task packet
- acceptance governance
- candidate promotion review

## Templates

- `expert-task-packet.md`：leader 派发给专家 agent 的任务包模板。
- `acceptance-report.md`：leader 对专家返回进行验收、重试和仲裁的记录模板。
- `candidate-review-report.md`：leader 对项目候选 subagent 晋升进行审查的记录模板。
