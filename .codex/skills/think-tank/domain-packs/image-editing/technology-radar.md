# Image Editing Technology Radar

本文件记录图像编辑领域默认技术雷达。

## P0

| 技术方向 | 说明 |
|----------|------|
| AI 消除 / 修复 | 对象移除、背景补全、边缘自然性 |
| Neural Engine 加速 | Core ML 推理、端侧性能和功耗 |

## P1

| 技术方向 | 说明 |
|----------|------|
| Generative Inpainting / Outpainting | 生成式修图、画面扩展 |
| AI 人像分割 | 人像、头发、主体边缘和遮罩质量 |
| AI 降噪 / 超分 | 品质增强、低光修复 |

## P2

| 技术方向 | 说明 |
|----------|------|
| Metal GPU 实时处理 | 实时预览、滤镜链、性能 |
| 胶片模拟和预设系统 | 色彩科学、社区分享 |
| RAW 工作流 | 色彩管理、专业用户 |

## 调研要求

每个技术方向应尽量回答：

- 技术原理是什么
- 哪些竞品已经使用
- Apple 生态是否支持
- 端侧实现难度
- 用户是否感知到价值
- 对产品路线的优先级建议
