# Support

`think-tank` is an open-source, protocol-first Skill.

## Best-Effort Support

The public repository supports:

- understanding the Skill core
- running release checks
- using public examples and templates
- contributing protocol, schema, template, and documentation improvements
- reporting boundary, privacy, or evidence-claim problems

## Not Included

The public repository does not provide:

- guaranteed provider setup
- user account login help
- private workspace debugging
- closed-source runtime support
- production support contracts
- help recovering secrets or private data accidentally committed elsewhere

## Before Asking For Help

Please run:

```bash
python3 checks/open_source_release_suite.py
python3 checks/stable_release_check.py
```

Include the command output, your platform, and the exact file or protocol area involved.
