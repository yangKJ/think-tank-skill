# Research OS Starter Kit

This starter kit can be copied into a user's project as a local `.think-tank/` workspace.

The templates are public-safe. Real workspace data is user-owned and should not be committed by default.

## Layout

```text
.think-tank/
├── inbox/
├── sources/
│   └── ledger.example.jsonl
├── artifacts/
├── decisions/
├── experiments/
├── runbooks/
├── memory/
├── runs/
└── provider-ledger/
```

## Copy Guidance

Copy the contents of this directory into a local project workspace only when the user wants a project-local Research OS.

Do not copy private data back into the public `think-tank` repository.
