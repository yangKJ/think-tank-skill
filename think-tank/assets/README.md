# Think-Tank Visual Assets

这里存放 README 和公开文档使用的视觉资产。

## Governance

- Use PNG images for README and documentation hero/card visuals.
- Do not add SVG primary display assets.
- Every README image link must point to a tracked file under `think-tank/assets/`.
- 生成式视觉资产应在 `prompts/` 记录 source prompt。
- 不要把私有项目截图、本地运行 artifact、凭据或用户特定资料放在这里。
