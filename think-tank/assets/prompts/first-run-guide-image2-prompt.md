# Image2 Prompt: First Run Guide

Create a clean documentation visual for the open-source AI skill "think-tank"
that explains a user's first successful run.

Visual goal:
- show that installation is already complete
- focus on the first meaningful use, not packaging
- make "boundary-aware output" feel approachable

Composition:
- left: a simple prompt card with the sentence "Use think-tank to review these notes"
- center: a visible flow from "prompt" -> "structured reasoning" -> "recommendation"
- right: output blocks labeled "facts", "assumptions", "risks", "boundary"
- add a small status cue: "public skill core"
- add one subtle callout: "selected is not invoked"

Style:
- documentation-friendly product illustration
- light theme
- realistic software UI framing, not abstract diagram art
- restrained blue-green accents
- large readable labels
- no people
- no dark background

Aspect ratio: 16:9.
