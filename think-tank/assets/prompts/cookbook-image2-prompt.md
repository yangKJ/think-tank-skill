# Image2 Prompt: Cookbook

Create a scenario-based documentation visual for the open-source AI skill
"think-tank" showing four typical usage patterns.

Visual goal:
- make the cookbook immediately scannable
- communicate "different task shapes, same core skill"

Composition:
- a 2x2 grid of scenario panels
- panel 1: "code review"
- panel 2: "market research"
- panel 3: "strategy discussion"
- panel 4: "provider-assisted research"
- each panel should look like a realistic product workflow card
- add small summary tags like "risks", "evidence", "tradeoffs", "boundary"

Style:
- premium documentation illustration
- light theme
- visually distinct panels with shared brand language
- no tiny text
- subtle product UI realism
- no human characters

Aspect ratio: 16:9.
