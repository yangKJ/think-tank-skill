# Image2 Prompt: Progression Guide

Create a documentation visual for the open-source AI skill "think-tank" that
shows how a user progresses from beginner to advanced usage.

Visual goal:
- communicate capability growth
- show that the upper bound comes from better operation, not just installation

Composition:
- a left-to-right progression path with three stages:
  "beginner", "intermediate", "advanced"
- beginner stage shows first run and structured output
- intermediate stage shows mode choice and provider boundary awareness
- advanced stage shows policy tuning, composition, and evidence discipline
- include an upward cue for "upper bound"
- include a small note: "better operation unlocks more value"

Style:
- realistic product roadmap / maturity visual
- light theme
- documentation-friendly
- strong hierarchy and readable labels
- confident but not flashy

Aspect ratio: 16:9.
