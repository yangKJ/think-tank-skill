# Image Editing Competitors

本文件记录图像编辑领域的默认竞品分类。

这些竞品来自旧 research agent 经验，只作为 image-editing domain pack 的默认上下文，不属于 think-tank core。

## 国际竞品

| 竞品 | 关注点 |
|------|--------|
| Adobe Lightroom | 专业修图、订阅、Raw、AI 降噪 |
| VSCO | 滤镜、胶片感、社区、订阅 |
| Snapseed | 易用性、Google 图像算法、免费策略 |
| Afterlight | 滤镜数量、简洁编辑 |
| Photomator | Apple 生态、AI 增强、专业编辑 |
| Darkroom | iOS 原生体验、视频和照片编辑 |

## 国内竞品

| 竞品 | 关注点 |
|------|--------|
| 醒图 | AI 消除、模板、年轻用户、内容传播 |
| 美图秀秀 | 用户规模、AI 美颜、功能广度 |
| 像素蛋糕 | 人像精修、专业工作流 |
| 泼辣修图 | 专业调色、口碑、社区 |

## 新兴或相邻产品

| 竞品 | 关注点 |
|------|--------|
| PicsArt | AI 艺术生成、模板和社区 |
| Photoroom | AI 背景移除、产品图 |
| Fotor | 多平台 AI 编辑 |
| TouchRetouch | 对象移除、修复体验 |
| Photoleap | 生成式 AI 和创意编辑 |
