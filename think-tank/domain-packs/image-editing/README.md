# Image Editing Domain Pack

该领域包承接旧 research agent 中关于图像编辑 App、竞品研究、新技术调研和用户舆情的经验。

它是可选领域包，不属于 think-tank core。

## 适用场景

- 图像编辑 App 竞品研究
- AI 修图技术调研
- iOS 图像处理技术 scouting
- 社媒用户反馈分析
- 产品路线和功能优先级讨论

## 默认关注点

- 竞品功能和定位
- 新技术信号
- 用户反馈和痛点
- 商业模式和订阅策略
- iOS / Apple 生态可实现性
- 图像处理性能和体验

## 与 core 的关系

```text
think-tank core:
  定义如何研究、讨论、审议、汇总

image-editing domain pack:
  定义图像编辑领域默认研究什么、看哪些来源、用哪些模板
```

## 文件

- `competitors.md`：默认竞品和分类
- `technology-radar.md`：默认技术雷达
- `source-map.md`：默认信息源
- `report-templates.md`：领域报告模板
