# v0.1 Readiness

本文记录 think-tank `0.1.0` 基础版本的验收状态。

## 版本定位

`0.1.0` 是 foundation 版本。

它证明：

- 主仓结构成立
- 协议边界成立
- Codex 单 agent 多 profile 执行路径可验证
- Codex 可以作为当前主工作平台运行 think-tank
- 旧 research agent 和 agent-council 已被归位为迁移素材
- 外部 skills 和旧 subagents 已通过 capabilities/profiles 抽象

它不声明：

- Claude Code Agent Team 已完成
- 真实多 agent 并发已完成
- 外部 skills 已集成测试
- Browser 外部网页 DOM 回收能力
- 任意领域 pack 都已经成熟

## 发布判断

当前可以作为公开 foundation 版本发布，但应明确写成：

> think-tank `0.1.0` 是协议和架构基础版本，Codex 主平台路径已完成 foundation 验收；Claude Code runtime、真实多 agent 和高风险外部 skills 的真实集成仍在 planned 状态。

不应宣传为：

- 已完成真实多 agent runtime
- 已完成 Claude Code Agent Team
- 已内置第三方工具型 skills
- 已支持所有领域
- 已完成 Browser 外部网页 DOM 回收
