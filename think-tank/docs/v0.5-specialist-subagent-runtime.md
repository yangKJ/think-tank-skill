# v0.5 Specialist Subagent Runtime

本文定义 think-tank `0.5` 阶段：专业 subagent runtime。

## Goal

```yaml
version_track: v0.5
theme: specialist_subagent_runtime
status: implemented_contract_and_runtime_primitives
```

v0.5 解决的问题：

> 单 agent 多 profile 只能模拟多视角，不能替代独立专业 subagent 的上下文隔离、工具链差异和结果回收。

因此 v0.5 增加专业 subagent 派发契约、role result schema、runtime helper、平台适配说明和 fallback 规则。
