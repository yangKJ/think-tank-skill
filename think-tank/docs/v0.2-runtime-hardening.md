# v0.2 Runtime Hardening

本文定义 think-tank `0.2` 阶段的建设范围。

## Goal

```yaml
version_track: v0.2
theme: runtime_hardening
source_reference: old_research_think_tank_runtime
status: implemented_minimal
```

v0.2 不扩展品牌和商业化方向，只吸收旧 research think-tank 中仍有价值的 runtime 工程经验。

## Non Goals

- 不恢复旧 research 父级关系。
- 不全量搬运旧 scripts。
- 不把 Claude Code Team 写进 core protocol。
- 不把旧 `skill.yaml` 变成新配置中心。
- 不把历史 `.think-tank/` run 结果当协议真相源。

## Verification

v0.2 新增检查：

```bash
python3 checks/runtime_contract_check.py
python3 checks/slot_contract_check.py
python3 checks/consensus_contract_check.py
python3 checks/research_protocol_check.py
python3 checks/runtime_planner_check.py
python3 checks/slot_resolver_check.py
python3 checks/state_model_check.py
python3 checks/consensus_runtime_check.py
```

前四个检查证明 v0.2 contract 已写入主仓，后四个检查证明平台无关 minimal runtime library 可运行。

仍不证明：

- Claude Code Team 真并发。
- 完整 adapter dispatch runtime。
- 自动 result recovery。
- 外部工具链 fallback。
