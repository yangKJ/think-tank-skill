# v0.2 Runtime Hardening

本文定义 think-tank `0.2` 阶段的建设范围。

## Goal

```yaml
version_track: v0.2
theme: runtime_hardening
source_reference: old_research_think_tank_runtime
status: implemented_as_contracts
```

v0.2 不扩展品牌和商业化方向，只吸收旧 research think-tank 中仍有价值的 runtime 工程经验。

## Completed Contracts

```yaml
runtime_contract:
  file: protocol/runtime-contract.md
  source_reference: core/workflow.py

slot_contract:
  file: capabilities/slot-contract.md
  source_reference:
    - runtime/slot_resolver.py
    - runtime/slot_validator.py

state_result_contract:
  file: protocol/state-result-contract.md
  source_reference: core/state.py

consensus_contract:
  file: protocol/consensus-contract.md
  source_reference: core/consensus.py

research_hardening:
  file: modes/research.md
  source_reference: references/research-protocol.md
```

## Non Goals

- 不恢复旧 research 父级关系。
- 不全量搬运旧 scripts。
- 不把 Claude Code Team 写进 core protocol。
- 不把旧 `skill.yaml` 变成新配置中心。
- 不把历史 `.think-tank/` run 结果当协议真相源。
