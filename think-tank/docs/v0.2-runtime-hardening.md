# v0.2 Runtime Hardening

本文定义 think-tank `0.2` 阶段的建设范围。

## Goal

```yaml
version_track: v0.2
theme: runtime_hardening
source_reference: old_research_think_tank_runtime
status: implemented_minimal
```

v0.2 不扩展品牌和商业化方向，只吸收旧 research think-tank 中仍有价值的 runtime 工程经验。

## Non Goals

- 不恢复旧 research 父级关系。
- 不原样搬运旧 scripts；可复用部分已经抽象到 runtime、templates、docs 和平台适配文档。
- 不把 Claude Code Team 写进 core protocol。
- 不把旧 `skill.yaml` 变成新配置中心。
- 不把历史 `.think-tank/` run 结果当协议真相源。


前四个检查证明 v0.2 contract 已写入主仓；runtime 检查证明平台无关 minimal runtime library 可运行；
adapter 检查证明 Codex adapter pipeline、Claude Code adapter spec、统一 runtime schema 和 E2E fixture 已接入；
最后三个检查证明旧 think-tank 的安全、模板和迁移处置已经纳入主仓。

仍不证明：

- Claude Code Team 真并发。
- 完整 adapter dispatch runtime。
- 自动 result recovery。
- 外部工具链 fallback。
