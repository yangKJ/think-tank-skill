# v1.0.0 Release Notes

本文是 `think-tank` 首个 stable release 的发布说明草稿，可直接作为 GitHub Release 描述基础。

## Release Summary

```yaml
version: v1.0.0
release_posture: stable_release
default_public_release: skill_core_only_bundle
default_platform: codex
stable_scope:
  - protocol-first high-level Skill
  - Codex-first verified foundation
  - explicit capability evidence states
  - stable release with scoped provider boundaries
```

## What Is Stable In v1.0.0

- `think-tank` core protocol、mode、profile、routing、runtime provenance 和 schema 已形成可检查的公开结构
- Codex 主路径已具备可重复通过的 foundation 验证
- external browser readonly、provider invocation matrix、beyond-readonly scoped subagent runtime、long-running subagent lifecycle 已有公开证据
- 仓库默认公开包已收口为 `skill_core_only_bundle`
- release privacy、package boundary、open source release、stable release 都有本地门禁

## Explicit Boundaries

`v1.0.0` 的 stable 不是“所有 optional provider 默认可用”，而是：

- stable protocol surface
- stable Codex-first default path
- stable evidence-based capability claims
- stable release governance

以下内容仍然必须按显式边界理解：

- optional providers 仍需按 provider 单独验证
- 外部网页登录态、复杂 browser interaction、社媒抓取、私有知识库写入不属于默认 stable 能力
- Claude Code runtime 仍是 `deferred`
- 跨平台 parity 仍需继续补强
