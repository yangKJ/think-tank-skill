# v0.3 Research Agent Migration

本文定义 think-tank `0.3` 阶段：旧 research agent 全仓迁移收口。

旧来源：

```text
legacy research workspace
```

## Goal

```yaml
version_track: v0.3
theme: research_agent_full_migration
status: complete
migration_style: inventory_and_abstraction
```

v0.3 的目标不是把旧 research agent 目录复制进主仓，而是做到：

- 旧 agents 全部映射到 profiles。
- 旧 skills 全部映射到 capability、platform adapter 或 out-of-core。
- 旧 private/domain knowledge 不进入当前主仓；由具体项目自行维护。
- 旧 memory、logs、run artifacts 全部明确处置。
- 旧平台私有配置全部排除出 core。
